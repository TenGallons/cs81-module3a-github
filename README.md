# cs81-module3a-github

## journal.js functions

- **greet(name):** Returns a greeting string using the provided name.
- **isEven(n):** Returns true if `n` is even, otherwise false.
- **fahrenheitToCelsius(f):** Converts Fahrenheit to Celsius, rounded to 1 decimal.
- **countVowels(text):** Counts vowels (a, e, i, o, u) in a string.
- **maxInArray(nums):** Returns the largest number in a non-empty array.
